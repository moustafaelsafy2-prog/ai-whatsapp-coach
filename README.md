# WhatsApp AI Bot

هذا مشروع bot للواتساب يمكنه:
- استقبال وإرسال الرسائل النصية
- معالجة الصور والفيديوهات والصوت
- قراءة وفهم المحتوى

## المتطلبات
- Node.js v14 أو أحدث
- حساب WhatsApp

## التثبيت
```
npm install
```

## كيفية التشغيل
1. قم بتثبيت الحزم:
   ```
   npm install
   ```

2. شغّل البوت:
   ```
   npm run bot
   ```

3. عند التشغيل الأول، ستظهر QR code في الطرفية. امسحها باستخدام تطبيق WhatsApp على الهاتف لتسجيل الدخول.

## المميزات
- **قراءة النصوص**: يستقبل ويقرأ الرسائل النصية
- **معالجة الوسائط**: یمكنه استقبال الصور والفيديوهات والملفات الصوتیة
- **الإجابة**: یهيئ لدمج AI للإجابات الذكیة (يمكن إضافته لاحقاً)

## ملاحظات مهمة
- لا تشغل البوت في بيئات إنتاج بدون تحسينات أمان إضافية
- تأكد من اتباع سياسات WhatsApp عند الاستخدام

## التطوير
لإضافة مميزات AI، قم بتعديل الملف `bot.js` لدمج API مثل OpenAI أو Google Vision.
